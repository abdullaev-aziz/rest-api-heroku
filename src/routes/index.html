rest-api-heroku

Basic Rest API supporting CRUD operations

Tech Stack

Node.js/Express/MongoDB & deployed to Heroku

Source Code

https://github.com/abdullaev-aziz/rest-api-heroku

Routes:

The default URL:

  • https://restapi-express.herokuapp.com/

Fetching all posts:

  • https://restapi-express.herokuapp.com/posts
    Supported Operations: GET

Fetching a specific post by providing ID:

  • https://restapi-express.herokuapp.com/posts/:id
    Supported Operations: GET, POST, DELETE, PATCH
